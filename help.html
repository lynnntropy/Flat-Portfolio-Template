Getting started

Thank you for purchasing the template! This guide is here to help you get started with customizing it to your preferences.

Replacing the cover page text

The current cover page of your portfolio is filled with Lorem Ipsum, the standard placeholder text that you're probably already familiar with.

You'll certainly want to replace it with your own elevator pitch to get people interested in your work, so here's how you can easily do that:

  • Open your index.html file, and find the part that says 'Who am I'.
  • Find the current placeholder text, right below that. It's between a pair of <p> tags and starts with 'lorem ipsum'.
  • Simply delete the placeholder text and replace it with your own.

Replacing the placeholder portfolio images

The images in the portfolio are currently public domain photographs used to demonstrate the functionality. Since you'll want to replace these with your own projects here's how:

  • Open index.html if you haven't already, and find the part that starts with
    <div class="projects panel-faded-out inactive-content">.
  • Find the elements that start with
    <div class="image-card card paper-shadow-top-z-1">. Each one of these is displayed as a portfolio image on your website.
  • For each of these elements, simply replace the image URLs (there are two for each image) with the location of your own image. This can be either an image hosted on your own website, or an external link (Imgur, Tumblr, Flickr, and so on).

Adding/removing projects from your portfolio

Even once you've replaced the images for each project, there's a slim chance you want exactly three projects in your portfolio. Follow this guide to add or remove projects from the portfolio to suit your needs.

  • Open index.html if you haven't already, and find the parts that start with
    <div class="projects panel-faded-out inactive-content">.
  • Each of these is displayed as a single project in your portfolio.
  • To delete the project, just delete the entire element, starting with the line you found and ending with '</div>'.
  • To add another project, copy the element (starting with the line you found and ending with '</div>') and paste it below or above the element. Then you can proceed to replace the text and images as shown above.

Enabling the contact form (advanced)

Since the template is a simple HTML/JS/CSS template with no server-side component, you will have to implement this functionality yourself if you want the contact form to work. Generally, you'll want the submit button to either submit the form's contents via HTTP POST (and then send that message somehow by catching it with serverside code, e.g. PHP), or send it to a third-party email-sending service via a clientside API (i.e. JavaScript code).

Since this is an advanced procedure intentionally left vague due to differences in implementation, there is no step-by-step guide.

Note: if you wish to remove the contact form so your visitors do no get confused by it, remove the element starting with
<div class="contact-form"> from your index.html.

Changing any other text content in the template

If you want to change any text on your website that isn't specifically covered by the guide, follow these steps:

  • Open index.html if you haven't already, and the text you want to change. Most text editors have a 'find' feature you can use to do this quickly.
  • Once you've found the text, select it in its entirety, and hit delete.
  • Fill the space left by the text you deleted with whatever text you wish to replace it with.
